# Marvelous-Todos
Mini project with React, Webpack and Firebase

## Hierarchy of the components:
- Header
- Section
  * ItemAction
    * AddItem
    * SearchItem
  * Column1
  * Column2

## How to try:
- clone repo:
```
git clone git@github.com:aindrayana/marvelous-todos.git
```
- install npm component
```
npm install
```
- run server
```
npm run dev
```
- run browser on http://localhost:3000/
